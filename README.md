# Code4Beginners – Meine erste Website

Eine interaktive Lern-Website für Marketing-Praktikanten, die das Programmieren lernen möchten.

## 📄 Seiten

- **Startseite** – Übersicht und Motivation
- **10 Levels** – interaktive HTML/CSS-Aufgaben von leicht bis schwer
- **HTML Grundlagen** – Lerne die wichtigsten HTML-Tags
- **CSS Grundlagen** – Lerne Farben, Layout und Design

## 🌍 Sprachen

Die Website unterstützt 5 Sprachen (umschaltbar über die Flaggen oben rechts):
- Deutsch
- English
- Español
- Français
- Italiano

## 🛠️ Verwendete Technologien

- HTML5
- CSS3
- JavaScript
- FlagCDN (Länderflaggen)
- Unsplash (Stock-Fotos)

## 🚀 Online ansehen

Einfach `index.html` im Browser öffnen – kein Server nötig!
