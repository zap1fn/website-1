Was ist CSS?

CSS (Cascading Style Sheets) ist die Gestaltungssprache des Webs. Während HTML die Struktur vorgibt, bestimmt CSS das Aussehen: Farben, Schriftarten, Abstände, Positionen.

Stell dir vor: HTML ist ein Haus (Zimmer, Wände, Türen), CSS ist die Inneneinrichtung (Farbe der Wände, Möbel, Deko).

So schreibt man CSS

CSS besteht aus Regeln. Jede Regel hat einen Selektor (wer?) und einen Block mit Eigenschaften (was?).

/* Alle h1 Überschriften werden blau */
h1 {
    color: blue;
    font-size: 32px;
}

Wichtige CSS-Eigenschaften

Diese Eigenschaften brauchst du am häufigsten:

  • color – Textfarbe
  • background – Hintergrundfarbe oder -bild
  • font-size – Schriftgröße
  • margin / padding – Abstände außen / innen
  • display: flex – Modernes Layout
